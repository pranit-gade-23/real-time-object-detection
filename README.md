# Real-Time Object Detection System

This project implements a real-time object detection system using TensorFlow and OpenCV.  
It detects objects from live webcam feed and video files using a pre-trained SSD MobileNet V2 model.

---

## 🚀 Features

- Real-time object detection using webcam
- Detection from video files
- Bounding boxes with labels and confidence scores
- Non-Max Suppression to remove duplicate detections
- Optimized frame processing for smooth performance

---

## 🛠️ Technologies Used

- Python
- TensorFlow
- TensorFlow Hub
- OpenCV
- NumPy
